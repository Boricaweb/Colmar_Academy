
# Colmar Academy Web

My own design website to be part of learning version control system with Git and GitHub. So, I created the Colmar Academy website to be an sample web for learning. As always, this project is part of Full-Stack Engineer course of www.codecademy.com.


## Lessons Learned

* CSS responsive web design with
  * clamp() function
  * min() function
  * max() function
* CSS nested
* CSS pseudo selectors
  * :is()
  * :where()


## Tech Stack

**Front-end:** HTML, CSS (responsive)
